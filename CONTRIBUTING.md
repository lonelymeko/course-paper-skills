# Contributing

Keep the skills concise and script-backed.

Rules:

- Do not add real user papers, reports, tokens, cookies, or QR login artifacts.
- Do not add instructions for bypassing captcha, payment, login, SMS, QR, or anti-bot systems.
- Save new provider details only after a real official response has been observed.
- Prefer bundled scripts for fragile flows and keep SKILL.md focused on workflow decisions.

Before opening a pull request:

```bash
npm run check
```
